## 📖 專案簡介

此網頁提供香港 10 條熱門行山路線的詳細資訊，包括難度等級、預計時間、路線距離、特色描述和注意事項。無論您是行山新手還是經驗豐富的登山愛好者，都能找到適合自己的路線。

## ✨ 功能特色

- **響應式設計**：完美支援桌面、平板和手機等各種設備
- **互動篩選功能**：按難度等級（初級、中級、高級、專家級）快速篩選路線
- **美觀的卡片式佈局**：每條路線以卡片形式展示，清晰易讀
- **平滑動畫效果**：卡片載入和切換時的淡入淡出動畫
- **詳細路線資訊**：包含時間、距離、特色和注意事項
- **安全提示和準備清單**：提供實用的行山建議

## 🗺️ 包含的行山路線

### 初級路線（2 條）
- 龍脊 (Dragon's Back) - CNN 評選亞洲最佳市區遠足徑
- 大潭水塘 - 歷史古蹟，適合親子同遊

### 中級路線（3 條）
- 獅子山 (Lion Rock) - 香港精神象徵
- 鳳凰徑第二段 - 沿海岸線美景
- 麥理浩徑第二段 - 穿越馬鞍山郊野公園

### 高級路線（3 條）
- 大帽山 (Tai Mo Shan) - 香港最高峰
- 蚺蛇尖 (Sharp Peak) - 香港三尖之首
- 麥理浩徑第三段 - 包含多個景點

### 專家級路線（2 條）
- 鳳凰山 (Lantau Peak) - 香港八景之一「鳳凰觀日」
- 西貢東全走 - 最具挑戰性的長途路線

## 🚀 如何使用

### 本地運行

1. 下載或克隆此專案：
```bash
git clone https://github.com/honglocki2005/hiking-hk.git
```

2. 在瀏覽器中打開 `index.html` 檔案即可瀏覽：
   - 直接雙擊 `index.html` 檔案
   - 或使用本地伺服器（推薦）

### 使用篩選功能

1. 頁面載入後，預設顯示所有路線
2. 點擊頂部的篩選按鈕（全部路線、初級、中級、高級、專家級）
3. 只會顯示符合選擇難度的路線
4. 卡片會以動畫效果淡入
